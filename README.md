## 📖 О проекте

**КиберКачалка** — это не просто сайт-визитка, а полноценная цифровая экосистема для уникального фиджитал-центра (физический + цифровой). Это место, где киберспорт (PS5, турниры) переплетается с реальным спортом (футбол, баскетбол, единоборства).

Проект представляет собой веб-платформу с атмосферой **киберпанка**, включающую в себя онлайн-запись, бонусную программу, админ-панель и новостную ленту.

## 🚀 Ключевые функции

### 👤 Пользовательская часть

| Функция | Описание |
|---------|----------|
| **Атмосферный UI/UX** | Неоновая эстетика, 3D-границы, плавающие частицы и плавные анимации |
| **Регистрация с верификацией** | Подтверждение email через SMTP (Mail.ru) с 6-значным кодом |
| **Умная запись (бронирование)** | Пошаговая форма с выбором услуг, поддержка физ. и юр. лиц, расчет стоимости на лету |
| **Личный кабинет** | История бронирований, статусы заявок (Новая → Подтверждена → Завершена) |
| **Бонусная система "КиберДруг"** | 5% баллов от чека, уровни лояльности, прогресс-бар |
| **Динамический прейскурант** | Цены из БД, автоматическая группировка, льготные категории |
| **Галерея новостей** | Карусель с последними событиями центра |

### 🛡️ Административная панель

| Функция | Описание |
|---------|----------|
| **Управление заявками** | Изменение статусов, просмотр деталей бронирования |
| **Начисление бонусов** | Ручное начисление баллов за выполненные заказы |
| **CRUD новостей** | Добавление и удаление новостей прямо из интерфейса |
| **Управление пользователями** | Просмотр и удаление пользователей |
| **Прайс-лист** | Просмотр всех услуг из базы данных |

## 🛠️ Технологический стек

### Backend

| Компонент | Технология |
|-----------|------------|
| **Язык** | Python 3.x |
| **Фреймворк** | Flask (Jinja2 templating, sessions) |
| **База данных** | Supabase (PostgreSQL client) |
| **Email-уведомления** | SMTP (Mail.ru) |
| **Хэширование** | hashlib (SHA-256) |
| **Токены** | itsdangerous (сброс пароля) |

### Frontend

| Компонент | Технология |
|-----------|------------|
| **Язык** | Vanilla JavaScript (ES6+) |
| **Разметка** | HTML5 |
| **Стили** | CSS3 (Flexbox, Grid, Keyframes, Filters) |
| **Иконки** | Font Awesome 6 |
| **Адаптивность** | Mobile First |

## ⚙️ Установка и запуск

### Предварительные требования

- Python 3.8+
- Аккаунт [Supabase](https://supabase.com/)
- SMTP-доступ (Mail.ru / Gmail / Яндекс)
